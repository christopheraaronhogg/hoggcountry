## Verification Checklist

When assessing AT content accuracy, check these sources:

### Mileage & Geography
- [ ] ATC Data Book (official distances)
- [ ] FarOut app (GPS-verified waypoints)
- [ ] AWOL Guide (yearly updates)

### Permits & Regulations
- [ ] Baxter State Park website (Katahdin rules)
- [ ] NPS GSMNP (Smokies permits)
- [ ] AMC website (White Mountain huts)

### Town Services
- [ ] FarOut user comments (real-time)
- [ ] The Trek resupply guide (updated regularly)
- [ ] Direct phone calls (most reliable)

### Trail Conditions
- [ ] FarOut comments (day-to-day conditions)
- [ ] WhiteBlaze forums (discussion threads)
- [ ] Trail club websites (maintenance updates)

### Hostel/Lodging
- [ ] Direct hostel websites
- [ ] The Trek hostel listings
- [ ] FarOut business listings
